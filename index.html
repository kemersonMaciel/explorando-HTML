O que é HTML?

HTML é a linguagem padrão utilizada para criar páginas web. Ele permite que os desenvolvedores estruturem o conteúdo de uma página, incluindo texto, imagens, vídeos e outros elementos, para que possam ser exibidos corretamente em um navegador.

O HTML é composto por uma série de elementos chamados de "tags".Cada tag tem uma função específica e ajuda a definir a estrutura e o significado do conteúdo dentro de uma página web

O que são Tags HTML?

As tags HTMLsão elementos utilizados para definir diferentes partes de uma página web. Elas são colocadas dentro do código HTML e envolvem o conteúdo que queremos que seja exibido de uma determinada maneira.

Existem diversas tags HTML disponíveis, cada uma com sua própria função e sintaxe. Abaixo terá algumas tags mais comuns:

  • <h1> até <h6>: São tags utilizadas para definir títulos de diferentes níveis de importância. O <h1> é o título mais importante e o <h6> é o menos importante.
  • <p>: É a tag usada para definir parágrafos de texto.
  • <mark>: Esta tag é usada para destacar parte do texto, geralmente com um fundo amarelo.
  • <small>: Define um texto como sendo de menor importância, geralmente exibido em um tamanho de fonte menor.
  • <i>: Utilizado para enfatizar texto, geralmente exibido em itálico.
  • <u>: Define texto sublinhado.
  • <strong>: Define texto importante, geralmente exibido em negrito.
  • <ol>: Define uma lista ordenada, onde os itens são numerados.
  • <ul>: Define uma lista não ordenada, onde os itens são marcados com pontos ou outros símbolos.
  • <li>: Utilizado dentro de <ol> ou <ul> para definir cada item da lista.
  • <a>: Utilizado para criar links para outras páginas.
  • <hr>: Cria uma linha horizontal para separar o conteúdo.
  • <sub>: Define texto como subscrito, ou seja, posicionado abaixo da linha de base.
  • <sup>: Define texto como sobrescrito, ou seja, posicionado acima da linha de base.
  • <blockquote>: Utilizado para citar um bloco de texto de outra fonte.
  • <font>: Esta tag costumava ser usada para definir a fonte e outras propriedades de texto, mas é considerada obsoleta em HTML5. É preferível usar CSS para estilizar o texto.
  • <del>: Define texto deletado, geralmente exibido com uma linha através dele.
  • <abbr>: Define uma abreviação ou um acrônimo, geralmente exibido com um estilo especial quando o mouse passa sobre ele.
